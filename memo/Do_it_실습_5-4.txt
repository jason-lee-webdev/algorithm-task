[문제]
void recur(n) {
    if(n > 0) {
        recur(n - 1);
        System.out.println(n);
        recur(n - 2);
    }
}
recur 메소드의 꼬리에서 재귀 호출하는 메소드 recur(n - 2)를
비재귀적으로 구현한다.

[로직]
recur(n - 2)는 인자를 n에서 n - 2로 변경 후 메소드를 다시 호출한다는 의미이다.
recur 메소드는 n > 0 일때 실행 블록 내의 코드를 실행한다.
반복문 안에서 n에  n - 2를 할당하고 메소드 실행 블록의 시작점으로 이동한다.
반복문은 n > 0 일때만 반복한다.
